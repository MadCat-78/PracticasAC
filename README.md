# Arquitectura de computadoras

Repositorio dedicado a almacenar todas las tareas,  realizadas durante el curso de la materia.

---

## Descripción

Este repositorio reúne el trabajo desarrollado a lo largo del semestre, incluyendo implementaciones, reportes, documentación y archivos relacionados con las actividades de la materia.

El propósito principal es mantener una organización clara del contenido y servir como respaldo académico.

---

## Estructura del Repositorio

```bash
📂 DOCS/
📂 RTL/
📄 README.md
```

DOCS contiene:
* Reportes
* Simulaciones
* Capturas o resultados

RTL contiene:
* Código fuente
* Archivos auxiliares
